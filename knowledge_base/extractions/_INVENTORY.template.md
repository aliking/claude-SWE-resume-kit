# Extraction Inventory

> Updated automatically by `/setup-extract`. You can also edit manually.

## How to use

1. Run `/setup-extract path/to/source.md` for each project, initiative, or source document
2. For stronger evidence, pass mixed sources for the same work item (resume PDF + repo path + annual review PDF)
3. Confirm merge/split decisions when the candidate map is shown
4. Each extraction creates a file in this directory
5. After all extractions, run `/setup-build-kb` to synthesize into experience files, bundles, etc.
